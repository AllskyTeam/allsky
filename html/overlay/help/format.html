Format Help

The Format field allows the data to be formatted using a coding system. The system is different for each type of field as detailed below.

  • Dates
  • Times

Date Type: Date

The date field format is used to format a date, please note that there is a seperate format option for times. The following codes are available
Code Description
j Day of the month without leading zeros (1 to 31).
d Day of the month, 2 digits with leading zeros (01 to 31).
l (lowercase 'L') A full textual representation of the day of the week (Monday, Tuesday,...).
w Numeric representation of the day of the week (0=Sunday,1=Monday,...6=Saturday).
D A textual representation of a day, three letters.
S The English ordinal suffix for the day of the month (2 characters st, nd, rd or th. Works well with j).
m Numeric representation of a month, with leading zeros (01 to 12).
n Numeric representation of a month, without leading zeros (1 to 12).
F A full textual representation of a month, such as January - March.
M A short textual representation of a month, three letters (Jan - Dec).
Y A full numeric representation of a year, 4 digits (1999 OR 2022).
y A two digit representation of a year (99 OR 22)

Date Format Examples

NOTE: When creating formats any characters not recognised as a formatting character will simply be added to the output. This allows you to use delimiting characters in the format.

The below examples all use the 8th May 2022 as the source date

Format Description
d/m/y 08/05/22
dS F Y 08th May 2022

Date Type: Time

The time field format is used to format a time, please note that there is a seperate format option for dates. The following codes are available
Code Description
H 24-hour format of an hour with leading zeros (00 to 23).
g 12-hour format of an hour without leading zeros (1 to 12).
h 12-hour format of an hour with leading zeros (01 to 12).
a Lowercase Ante meridiem and Post meridiem (am or pm).
i Minutes with leading zeros (00 to 59).
s Seconds, with leading zeros (00 to 59).
c ISO 8601 date (eg: 2022-05-08T21:00:54.944Z).

Time Format Examples

NOTE: When creating formats any characters not recognised as a formatting character will simply be added to the output. This allows you to use delimiting characters in the format.

The below examples all use the 21:00 as the source date

Format Description
H:m:s 21:00:00
g:ma 9:00pm
