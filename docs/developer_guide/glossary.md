# Glossary

| Item      | Description                          |
| ----------- | ------------------------------------ |
| Module       | Python code used to extend the functionality of Allsky  |
| Event      | An event, daytime / nighttime image capture, day to night or night to day transition, periodic |
| Flow       | A series of modules grouped together and triggered by an event |
| Pipeline    | Same as a flow |
| Block | A small json file that defines the layout of variables on the overlay manager |
