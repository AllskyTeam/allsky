This documentation provides information for installing, configuring, and running the AllSky software, including describing what various features like overlays and keograms are, and how to troubleshoot and fix problems.

You can accesss this documentation via the WebUI's Allsky Documentation link or the Wiki link on any Allsky GitHub page.

Questions and Feature Requests

If you have a QUESTION or want to REQUEST A NEW FEATURE create a new Discussion by clicking on the Discussions link on any Allsky GitHub page. Please do NOT create an Issue.

Issues

If you have a problem first look in the "Documentation -> Troubleshooting" pages for a solution - most known problems are listed there. If you can't find a similar problem look in the Discussions (it may be a closed discussion). If you still can't find your problem, read the Reporting Issues page and then create a new Issue.
